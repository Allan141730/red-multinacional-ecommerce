# Red Empresarial Multinacional – Cisco Packet Tracer
Universidad Fidelitas | Enero 2026 – Abril 2026

## Descripción
Diseño e implementación simulada de una red corporativa multinacional 
para una empresa de e-commerce con presencia en Costa Rica, EE.UU., 
Asia y Medio Oriente. El objetivo fue garantizar alta disponibilidad, 
seguridad y escalabilidad para más de 2,387 usuarios finales.

## Características Técnicas
- SLA de 99% de disponibilidad
- Segmentación por VLANs por departamento:
  - Servicio al Cliente: 1,550 usuarios
  - Operaciones: 450 usuarios
  - Finanzas: 387 usuarios
- Enrutamiento dinámico con OSPF entre sedes internacionales
- VPN site-to-site con IPSec para comunicación segura entre oficinas
- Redundancia de enlaces y balanceo de carga
- Políticas de acceso diferenciadas por VLAN

## Herramientas Utilizadas
- Cisco Packet Tracer
- Cisco IOS
- Microsoft Visio (diagramación)

## Entregables
- Topología completa simulada en Packet Tracer
- Análisis de costos: modelos Cisco, cantidades y presupuesto total
- Documentación de políticas de acceso y segmentación
